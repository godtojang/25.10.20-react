초보 웹 개발자 · HTML/CSS 학습 중

사용자가 보기 편한 화면을 만드는 것에 관심이 많다 시맨틱 태그를 사용해서 구조를 명확하게 작성하려고 노력하고 있으며 간단한 레이아웃, 텍스트 스타일링, 배경, 그림자 효과 등을 구현할 수 있다

기술 스택 / 사용 가능 기술

HTML

  • 시맨틱 태그(header, nav, main, section, footer)를 활용해 의미 있는 구조를 작성할 수 있다
  • 이미지('img')와 링크('a'), 단락('p'), 인라인 텍스트('span')를 적절히 사용할 수 있다

CSS

  • 테두리(border), 배경(background), 글자 스타일(text, text-shadow), 박스 그림자(box-shadow)를 적용할 수 있다
  • 컨테이너 라는 개념을 이해하고, 최대 너비를 제한하고 가운데 정렬하는 레이아웃 박스를 만들 수 있다
  • display와 flex를 이용해 가로 배치, 중앙 정렬을 할 수 있다
